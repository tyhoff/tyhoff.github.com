To access the Purdue Dining Court XML API use this format

>http://api.hfs.purdue.edu/menus/v1/locations/\<Dining Court\>/MM-DD-YYYY

where `<Dining Court>` is from the list below

* Earhart
* Ford
* Hillenbrand
* Wiley
* Windsor
